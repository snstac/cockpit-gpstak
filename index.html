GPSTAK — Network GPS for TAK

Feed this device's GNSS position to ATAK (Settings → Device Preferences → GPS Preferences → External or Network GPS) and WinTAK. CoT position events go to the COT_URL; WinTAK can take raw NMEA.

Service

Configuration

Broadcast reaches every ATAK device on the subnet; use udp+wo://<tablet-ip>:4349 for a single device.
